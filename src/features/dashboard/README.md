# Dashboard Feature Module

This folder controls the analytics cockpit, tracking kanban, and scheduling integrations.

## Planned Stack
- Recharts (visual graphics)
- React Drag and Drop (Kanban interface)
- Prisma (Application CRM lifecycle)

## Directory Structure
- `/components`: Overview stats widgets, Kanban board columns, metric trend charts
- `/hooks`: useApplicationsTracker, useAnalytics
- `/services`: CRM updates and aggregations
- `/types`: Tracker and board definitions
