# Users Feature Module

This module represents profile settings, resume uploads, and student settings.

## Planned Stack
- AWS S3 (resume uploads)
- Prisma (user metadata)
- React Hook Form + Zod (profile edit)

## Directory Structure
- `/components`: Profile settings card, academic history details, resume uploader
- `/hooks`: useProfile, useSettings
- `/services`: Profile mutations, file upload signatures
- `/types`: User types
