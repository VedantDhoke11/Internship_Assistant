# Internships Feature Module

This folder contains components and logic related to internship feeds, job description rendering, and company profiles.

## Planned Stack
- PostgreSQL via Prisma (data persistence)
- React Query (caching and sync)
- Algorithmic matching calculations

## Directory Structure
- `/components`: Internship card, feed search filters, detail drawers
- `/hooks`: useInternships, useBookmark
- `/services`: Ingest and feed fetchers
- `/types`: Internship schemas
